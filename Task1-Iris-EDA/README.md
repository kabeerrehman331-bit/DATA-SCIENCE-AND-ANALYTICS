# Task 1: Exploring and Visualizing the Iris Dataset

## Objective
Understand how to read, summarize, and visualize a dataset using Python.

## Dataset
- **Name:** Iris Dataset
- **Source:** Built into Seaborn library
- **Size:** 150 rows, 5 columns

## Approach
1. Loaded the dataset using Seaborn
2. Inspected structure using .shape, .columns, .head()
3. Checked for missing values
4. Created visualizations:
   - Scatter plot to analyze relationships between variables
   - Histogram to examine data distribution
   - Box plot to detect outliers and spread of values

## Libraries Used
- Pandas
- Matplotlib
- Seaborn

## Results & Insights
- The dataset has 150 rows and 5 columns with no missing values
- Setosa species is clearly separated from the others based on petal length
- Petal length is the most useful feature for distinguishing species
- No significant outliers were detected
